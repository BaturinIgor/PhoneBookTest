1) Запуск тестов в терминале для ОС Windows:
    a. git clone https://github.com/BaturinIgor/PhoneBookTest.git
    b. cd phoneBook
    c. Запустить приложение phonebook.jar
    d. mvn clean install -U (для загрузки недостающих плагинов)
    e. mvn allure:serve (для запуска сервера с отчетом о тестировании)

P.S. Если ОС другая, то скачать драйвер по ссылке https://chromedriver.chromium.org/ и распаковать в папку проекта.
